---

#### <FileTreeFolderIcon /> `/e2e`

End-to-end tests folder for [Playwright](https://playwright.dev/) tests.

#### <FileTreeFolderIcon /> `/prisma`

[Prisma](https://www.prisma.io/) folder for database schema `schema.prisma` and seed scripts.

#### <FileTreeFolderIcon /> `/scripts`

Folder for scripts that will be auto executed on build or setup step.

#### <FileTreeFileIcon /> `/scripts/build-info.ts`

This scripts will be run on `postinstall` and will auto generate a `build-info.json` file with commit and date info that can be display in the application.

#### <FileTreeFolderIcon /> `/src`

Root folder of all the source code of the application. When importing files, you can use `@/` to reference this folder.

#### <FileTreeFolderIcon /> `/src/app`

Next router folder, all routes of the application are in this folder. See the [Next App Router documentation](https://nextjs.org/docs/app/building-your-application/routing) for more details.
